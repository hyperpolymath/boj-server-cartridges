= container-lsp — Container Language Server cartridge

Scaffolded cartridge for a Language Server Protocol implementation covering
*docker*, *podman*, *nerdctl*.

== State

*Status:* `scaffolded` (per `cartridge.json#/states[0]`).

The cartridge directory exists, the manifest declares the LSP capability
surface, and the source lineage is archived under `_source-archive/`. The
adapter (`adapter/`), Idris2 ABI (`abi/`), Zig FFI (`ffi/`), and panels
(`panels/`) are *empty* — to be filled in a follow-up PR.

== Source lineage

The original implementation is an Elixir LSP server living in the now-archived
`polystack/poly-container/lsp/` (1150 LOC across 11 modules):

----
adapters/behaviour.ex                   — 119 LOC
adapters/docker.ex                      — 216 LOC
adapters/nerdctl.ex                     — 215 LOC
adapters/podman.ex                      — 220 LOC
adapters/supervisor.ex                  — 27 LOC
lsp/handlers/completion.ex              — 43 LOC
lsp/handlers/diagnostics.ex             — 36 LOC
lsp/handlers/hover.ex                   — 50 LOC
lsp/server.ex                           — 153 LOC
poly_container_lsp/application.ex       — 22 LOC
poly_container_lsp.ex                   — 49 LOC
----

These files are preserved verbatim in `_source-archive/elixir-lib/` for the
port effort. They are not built by this cartridge.

== Port plan

. *adapter/server.ts* — Deno LSP framing (header-and-content message loop over
  stdio), routing to per-backend dispatchers.
. *adapter/handlers/* — Deno per-method handlers (`textDocument/diagnostic`,
  `textDocument/hover`, `textDocument/completion`,
  `workspace/executeCommand`).
. *adapter/backends/* — per-provider subprocess managers (docker, podman, nerdctl).
  The Elixir supervision tree collapses to Deno workers + an internal
  supervisor object; OTP guarantees become explicit error-handling boundaries.
. *ffi/zig/* — native helpers where Deno's process / IO surface is
  insufficient.
. *abi/idris2/* — formal ABI for the LSP request/response contract. Required
  if/when this cartridge promotes from `Ayo` to `Shield` tier.
