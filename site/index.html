    Contribute a cartridge

    All cartridges are open-source. The registry welcomes new MCP cartridges that follow the BoJ conventions.

    1. Fork the registry

    Fork hyperpolymath/boj-server-cartridges and create a branch for your cartridge.

    2. Add your cartridge

    Place your cartridge under cartridges/<category>/<bucket>/<name>-mcp/. Include a boj.toml manifest.

    3. Update the catalogue

    Run just catalog to regenerate site/catalog.json with your new cartridge entry.

    4. Open a pull request

    Open a PR against main. CI runs hypatia checks. Once merged, the registry updates automatically.

    About the registry

    The BoJ cartridge registry is the canonical source for MCP cartridges that extend the Bundle of Joy server.

    Tiers

    Ayo

    Production-grade cartridges. Stable API, passing hypatia checks, and active maintenance.

    Teranga

    Experimental cartridges. Functional but subject to breaking changes while the API settles.
