= reposystem-mcp / adapter — REST/gRPC/GraphQL bridge
:orientation: deep

== Purpose

Exposes the (stub) reposystem FFI over three protocols: REST on **9130**,
gRPC-compat on **9131**, GraphQL on **9132**. Stateless.

== Files

[cols="1,4"]
|===
| File | Role

| `build.zig`                             | Zig build graph.
| `reposystem_adapter.zig`                | Protocol dispatch. Tools: `reposystem_list_repos`, `reposystem_check_health`, `reposystem_sync_mirrors`, `reposystem_run_audit`.
| `SIDELINED-reposystem_adapter.v.adoc`   | zig predecessor. Not built.
|===

== Invariants

* **Stateless.**
* Responses reflect whatever the (stub) FFI returns — the adapter does not
  embellish.
