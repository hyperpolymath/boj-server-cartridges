== How to use this

Each prompt is **self-contained** — a fresh agent can pick one up cold,
read it, and start working without further context. The structure is:

. *Goal* — one-paragraph problem statement.
. *Context* — key files, PRs, ADRs to read first.
. *Concrete steps* — numbered checklist.
. *Acceptance criteria* — definition of done.
. *Pitfalls* — gotchas surfaced during the original session.
. *References* — memory entries + standing policies.

Reload the
`session_2026_05_26_cartridge_extraction.md` memory entry for the wider
session narrative; prompts deliberately don't repeat what that captures.
